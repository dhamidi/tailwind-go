# Welcome to the Blog
tags: announcements, meta

This is the inaugural post on our new blog built with **tailwind-go**.

We built this blog to showcase what's possible with a pure-Go implementation
of Tailwind CSS. No Node.js, no npm, no build step — just Go.

## What Makes This Different

Unlike traditional Tailwind setups, this blog generates its CSS entirely in Go:

- Zero JavaScript build tools required
- CSS is generated at startup by scanning templates
- Content-hashed URLs for perfect caching
- Server-side rendering with `html/template`

## What's Next

Stay tuned for more posts about building modern web applications with Go
and tailwind-go. We'll cover dark mode, responsive design, and more.
